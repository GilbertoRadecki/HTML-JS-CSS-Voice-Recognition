# HTML-JS-CSS-Voice-Recognition
Formulário Criado para permitir o preenchimento usando reconhecimento de voz

Preenchimento de formulários on-line constantemente fracassam em virtude da falta de completude nas informações prestadas.
Permitir a utilização de recursos de transcrião de voz para texto (Speech to Text) cria um incentivo ao preenchimento mais completo, na medida que falar é mais natural que digitar.
